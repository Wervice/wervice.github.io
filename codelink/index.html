Codelink is a set of cool tools for devs

The Idea

Codelink is a project and collection of cross-platform applications built for developers. They support the process of developing, manageing and designing.
Codelink integrates in the workflow and OS easily and are very fast.

Features

FileTree

Besides the implementation of already existing software, Codelink has its own inteligent filemanger, focused on coding and getting simple, but anoying task done quickly.

It can do lots of things:

  • Own Command line
  • Previews
  • List view
  • Terminal support
  • Compression
  • Encryption
  • And so much more

The design is clean and intutive.



Notes

Besides the implementation of already existing software, Codelink has its own inteligent filemanger, focused on coding and encryption. It lets you do all things like Copy, Move,... and also gives you extra features like encryption and terminal launchers.

Codelink Browser

This is a specialized browser for fixing bugs and errors. It also has a research assistant build-in that helps getting better Google results. The basic tools of any browser, like bookmarks are also included. It uses the Chromium rendering engine (webview).

Codelink Links

There are dozens of programs out there that devs use. I wont replace (them all), but you can run them easily by selecting a code editor, terminal and filemanager.

It's open source
