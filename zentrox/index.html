Finishing the installation

In the following, I will explain how you can finish the installation of Zentrox.

1. Connect to your server

To set Zentrox up, you have to get a shell/terminal session on your Linux server.
This can be achieved using physical access or SSH ('Secure Shell' is a remote control protocol).
You will need a terminal connection either using SSH or a terminal app on your server.

2. Paste the copied command

You can now paste the command you just copied or enter it manually into your terminal.
The command is as follows:
curl -fsSL https://raw.githubusercontent.com/Wervice/zentrox/main/install.bash -o zentrox_installer.bash; bash zentrox_installer.bash

3. Follow the installer

The Zentrox installer will now start.
Please follow the instructions in your terminal to finish the setup.

4. Start Zentrox server

The installer will provide you with another command.
Please copy this command and save it somewhere where you can retrieve it.
You can then paste the command to your terminal and run it.
In case you didn't get the command, here is it:
cd /home/YOUR_USERNAME/zentrox; node index.js
Please replace "YOUR_USERNAME" with your actual username.
If you do not know your username, you can get it by running the command whoami in your terminal.
Your Zentrox instance can now be accessed on https://IP_ADDRESS_OF_YOUR_HOME_SERVER:3000
Please replace "IP_ADDRESS_OF_YOUR_HOME_SERVER" with the actual IP adress of your server.
If you do not know this, you can use the command ip addr.

Zentrox

Easy home server admin and setup

Get started right from the terminal ðŸ‘‡

Zentrox is an open-source project that enables server remote control and home server setup through an intuitive web interface. It includes integrated tools for easy management. With Zentrox, you can efficiently manage and maintain your servers from any location.

Servers

Zentrox supports basic and advanced servers and options like FTP and SMB by default. They can be configured and started using the web interface. Every server has integrated encryption with easy setup.

Supported servers:
FTP | (Soon to be) in work: SMB, WebDAV, SSH

Resources

Monitor your system's resources with an intuitive dashboard. Watch real-time CPU, RAM, and disk usage, track running processes, and more. Stay in control with easy-to-read metrics.

Measured resources:
CPU, RAM, System disk, Battery state, System name

Open source

Zentrox is a fully open source and free project. It has no interest in tracking you or collection your data and also does not do so. You can view the code on the GitHub repository.

GitHub repository:
https://github.com/wervice/zentrox

Packages

You can manage system packages & applications using a simple interface that supports most Linux package managers. It lets you search, remove and install tools onto your system with just a few clicks.

Supported package managers:
DNF (RHEL/Fedora), APT (Debian), Pacman (Arch)
